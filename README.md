🧠 VitalAI - Smart Healthcare Assistant

VitalAI is an AI-powered healthcare assistant that helps users analyze symptoms, check medicine information, and take quick action in emergencies.

---

🚀 Features

- 🤖 AI Chat for symptom analysis
- 🚨 Emergency quick alert
- 🎤 Voice input support
- 📍 Find nearby hospitals (via Google Maps)
- 💊 Medicine safety checker
- 📄 Download chat report

---

🛠️ Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- AI API: Claude (Anthropic API)

---

📦 Installation

git clone https://github.com/YOUR_USERNAME/vitalai.git
cd vitalai
npm install

---

▶️ Run the App

node server.js

Open in browser:

http://localhost:5000

---

🌐 Deployment

You can deploy this app using:

- Render
- Railway
- Vercel (frontend) + backend services

---

🔐 Environment Variables

Create a ".env" file:

CLAUDE_API_KEY=your_api_key_here
